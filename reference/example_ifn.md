# Example of distributed forest inventory stands

An example of an coordinates, topography, forest and soil data
corresponding to 30 forest inventory plots.

## Format

The data format is that of an object
[`sf`](https://r-spatial.github.io/sf/reference/sf.html)

## Source

- Soil data from SoilGrids global database (Hengl et al. 2017).

- Soil depth and depth to bedrock from Shangguan et al. (2017).

- Forest structure and composition from the Third Spanish Forest
  Inventory (IFN3).
